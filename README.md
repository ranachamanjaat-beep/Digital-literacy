# Digital-literacy
This project for VIT Bhopal’s CSE0001 Digital Literacy course requires building a GitHub portfolio. Students must complete five tasks: creating an infographic, setting up professional profiles, practicing coding, drafting emails, and researching cybercrime. The final 100-mark submission includes a repo and report.
